# Web GIS 基礎

## ArcGIS Online へアクセス
1. [ArcGIS Online](https://www.esrij.com/products/arcgis-online/) にアクセスしてください。

   <img src="./img/agol.png" width="500px">

2. ログイン情報を入力し、ArcGIS Onlineにログインしてください。

   <img src="./img/agol_login.png" width="300px">

## Living Atlas へアクセス
1. 画面上部にある赤枠のボタンを押下してください。
 
   <img src="./img/agol_menu.png" width="500px">

2. 「Living Atlas」ボタンを押下します。

   <img src="./img/agol_living_atlas.png" width="300px">

3. Living Atlas 表示後、「地震」と入力し、検索を実行します。

   <img src="./img/living_atlas.png" width="500px">

## Web マップ参照
1. 検索結果の表示後、「南海トラフ巨大地震の被害想定（震度/最大クラス）」を押下します。

   <img src="./img/living_atlas_search.png" width="500px">

2. 画面遷移後、サムネイルを押下します。

   <img src="./img/living_atlas_earthquake.png" width="500px">

3. 「南海トラフ巨大地震の被害想定（震度/最大クラス）」 Web マップが起動します。

   <img src="./img/agol_earthquake.png" width="500px">
